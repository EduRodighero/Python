# 🐍 Estudos em Python

Este repositório reúne exercícios, exemplos e pequenos projetos que desenvolvo enquanto estudo **Python**.  
Um espaço para praticar, aprender e evoluir 🚀

---

## 🎯 Objetivo
Explorar a linguagem Python desde os fundamentos até tópicos avançados, consolidando o aprendizado com prática.

---

## 📂 Conteúdo
- **Fundamentos** → variáveis, estruturas de controle, funções  
- **Intermediário** → módulos, arquivos, exceções  
- **Avançado** → POO, geradores, decorators  
- **Projetos** → aplicações práticas
